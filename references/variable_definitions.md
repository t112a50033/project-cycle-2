# Variable Definitions

## SadOrHopeless
- Variable name: SadOrHopeless
- Meaning: Whether the student felt sad or hopeless
- Valid codes:
  - 1 = Yes
  - 2 = No
- Recoding:
  - 1 → 1 (Yes)
  - 2 → 0 (No)
- Benchmark:
  - p0 = 0.30
